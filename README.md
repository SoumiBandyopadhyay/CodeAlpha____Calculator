# 🚀 Space-Themed Calculator

A sleek and interactive calculator built using **HTML, CSS, and JavaScript**. The application performs basic arithmetic operations and features a modern **space-themed design** with an animated starry background, glassmorphism effects, and keyboard support.

---

## 📌 Project Overview

The Space-Themed Calculator is a front-end web development project designed to demonstrate the integration of HTML, CSS, and JavaScript in creating a functional and visually appealing calculator.

Users can perform basic mathematical calculations through both on-screen buttons and keyboard inputs. The project focuses on clean UI design, responsive layouts, and interactive functionality.

---

## ✨ Features

### 🧮 Calculator Operations
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Decimal number support

### 🎨 User Interface
- Space-themed animated background
- Glassmorphism-inspired design
- Responsive layout for different screen sizes
- Smooth hover effects and transitions
- Clear and intuitive button arrangement

### ⚡ Functionality
- Real-time user input handling
- Calculate results instantly
- Clear display functionality
- Backspace/Delete functionality
- Error handling for invalid expressions

### ⌨️ Keyboard Support
- Number keys (0–9)
- Arithmetic operators (+, -, *, /)
- Decimal point (.)
- Enter key to calculate
- Backspace key to delete
- Escape key to clear display

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser such as:

- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari

### Installation

#### Clone the Repository

```bash
https://github.com/SoumiBandyopadhyay/CodeAlpha____Calculator
```

#### Navigate to the Project Directory

```bash
cd space-calculator
```

#### Run the Project

Open the `index.html` file in your browser.

## 🎮 Usage

### Using the Mouse

1. Click number buttons to enter values.
2. Select an operator (+, -, ×, ÷).
3. Press `=` to calculate the result.
4. Use `C` to clear the display.
5. Use `⌫` to delete the last entered character.

### Using the Keyboard

| Key | Action |
|------|--------|
| 0–9 | Enter numbers |
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| . | Decimal point |
| Enter | Calculate result |
| Backspace | Delete last character |
| Esc | Clear display |
